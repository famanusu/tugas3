1. Persamaan Regresi Linear

Model regresi linear dinyatakan sebagai:

y = β₀ + β₁x

ΣX = 175, ΣY = 25, ΣXY = 975, ΣX² = 6375, n = 5

Hitung β₁:

β₁ = (nΣXY - ΣXΣY) / (nΣX² - (ΣX)²)

β₁ = (5×975 - 175×25) / (5×6375 - 175²)

β₁ = (4875 - 4375) / (31875 - 30625) = 500 / 1250 = 0.4

Hitung β₀:

β₀ = (ΣY - β₁ΣX) / n

β₀ = (25 - (0.4 × 175)) / 5

β₀ = (25 - 70) / 5 = -45 / 5 = -9

Persamaan Regresi:

Y = -9 + 0.4X
